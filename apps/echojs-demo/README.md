# echojs-demo

Минимальное приложение на [Vite](https://vitejs.dev/), чтобы проверить `echojs-jsx` в браузере.

```sh
# из корня репозитория
bun install
cd apps/echojs-demo
bun run dev
```

Откроется страница с примером из `packages/echojs-jsx/examples/usage-example.ts` (счётчик, input, `show`, `Fragment`).

Порт по умолчанию: **5174** (см. `vite.config.ts`).
