`@echojs/http` — HTTP клиент для приложений и библиотек, построенный вокруг:

- **immutable instances**
- **normalized options model**
- **hooks** (lifecycle без “магии”)
- **adapters** (честные capability boundaries)
- **strong error model** (классы ошибок, `instanceof`, type guards)

## Установка

Пока пакет `private` в монорепо, но публичная установка будет стандартной:

```bash
bun add @echojs/http
```

## Quick Start

```ts
import { createHttpClient } from "@echojs/http";

const http = createHttpClient({
  baseUrl: "https://api.example.com",
  timeout: { request: 10_000, response: 10_000, read: 30_000 },
  retry: { limit: 2 },
});

const me = await http.get("/me").unwrapJson<{ id: string }>();
```
